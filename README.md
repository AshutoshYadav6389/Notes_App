# Notes_App
A simple Notes Application built using HTML, CSS, and Vanilla JavaScript.
Users can add, save, delete, and persist notes using Local Storage.

Features
 Add new notes
 Save notes
 Delete notes
 Auto-save on focus out
 Notes stored in browser LocalStorage
 Clean and responsive UI

 Font Awesome icons
 Technologies Used
 
HTML5
CSS3
JavaScript (ES6)
Font Awesome
Local Storage
